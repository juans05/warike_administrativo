# Proyecto: Plataforma de Fidelización + Wallet + QR/NFC para Restaurantes (Warikes)

## Objetivo

Desarrollar una plataforma SaaS para restaurantes, cafeterías y warikes donde los clientes puedan acumular puntos o sellos digitales mediante QR o NFC, utilizando tarjetas digitales tipo Apple Wallet / Google Wallet.

La idea es reemplazar las tarjetas físicas de fidelización y convertirlo en un ecosistema de:

* fidelización
* marketing
* reseñas
* analítica
* automatización
* CRM gastronómico

---

# Flujo del cliente

1. Cliente compra en restaurante.
2. Restaurante muestra QR o NFC.
3. Cliente escanea.
4. Se abre tarjeta digital.
5. Sistema agrega puntos/sellos automáticamente.
6. Cliente desbloquea premios:

   * descuentos
   * productos gratis
   * promociones
   * niveles VIP

# NFC

Usar:

* NTAG213
* NTAG215
* NTAG216

Funcionamiento:

* cada mesa o cliente tiene un NFC
* al acercar el celular:

  * abre tarjeta
  * suma puntos
  * registra visita
